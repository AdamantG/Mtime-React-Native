# Mtime-React-Native
 
## 具体模块与组件使用
* 用React-Native实现时光网APP（包括正在热映、影片详情、演职员表、影片评论、预告片视频播放、剧照大图查看）
 
* 使用TabNavigator作为分页导航
 
* 使用FlatList进行列表展示，实现下拉刷新&上拉加载
 
* 使用Animated实现滑动时改变Title透明度的动画特效
 
* 使用React Native提供的Fetch API进行网络访问
 
* 引入react-native-video进行视频播放
